## What are the categories you are considering annotating?  These need not be fully finalized yet (AP2 will do that after you start digging into the data) but give us enough information to check if you're on the right track.

The annotation task that is posed is a multiclass noncompetitive classification. The classes will be *Loaded-Language*, *Reductio ad Hitlerum*, *Whataboutism*, *Red-herring*, *Bandwagon*, and *Strawman* as described in ([Vorakitphan, Cabrio, Villata (2021)](https://aclanthology.org/2021.clicit-1.53.pdf)). From the presence and number of categories that are relevant, you can create a measure of the proportion of statements (calculated as documents with presence/ number of total documents) that rely on logical fallacies and a measure for the prevalence of it in a statement (number of categories that apply/total number of categories).
